.. _Versioning:

Versioning scheme
=================

GEMPICX is _not_ performing semantic versioning.  
The versioning scheme does, however, consist of three numbers, ``Major.Minor.Patch``, and only applies to the master branch.  
 - ``Major`` and ``Minor`` are both taken from the latest tag of the release.
 - ``Patch`` refers to the number of commits to master since the latest tag.
If the repository is copied without git or shallowly fetched, no versionin information is available.
